苦工在兽人大军中的地位是最低的。由于他们对作战等重要技能一窍不通，只能来做诸如采伐木材、挖掘金矿等等下贱的工作。同时他们也负责建造和维修兽人大军所需的各种建筑物，他们夜以继日地工作以求能取悦他们的主人。

5个苦工就可以达到最大的采金效率，增加更多的毫无意义。

在基地遭受攻击的时候，苦工可以进入兽人地洞来进行防御。事实上，苦工是唯一可以进入兽人地洞的单位。

苦工的作用不止在于建造建筑或者收集资源，你同样可以把它们用在战斗上。你可以在基地对苦工使用里灵魂链接，让他们分担作战单位的伤害。人手不够时，也可以考虑用苦工来围杀敌人。但是不要用来对付那些有着强力范围杀伤的英雄，他们可能在瞬间就秒了你所有的苦工。
修理 Repair (自动技能)
当修理技能开启时，苦工可以自动修理建筑物和机械单位。修理需花费黄金和木材。如果你没有足够的黄金或木材将无法修理。

苦工可以去修理粉碎者，也可以修理同盟的机械单位或者建筑。
采集 Gather
从金矿中采集金钱以及从树林中砍伐树木。
返还资源 Return Resources
将携带的资源运送回最近的城镇中。
建造建筑 Build Structure
从可建造的建筑列表中选择你想要建造的建筑。
掠夺 Pillage (被动技能)
每次攻击建筑物时不仅造成伤害，而且还可以掠夺资源。这是一个被动技能。如果你毁掉了那个建筑，那么就可以掠夺回那个建筑造价的 50%。
