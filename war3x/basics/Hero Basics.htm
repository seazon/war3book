经验槽
英雄通过他或者他“指挥”的部队杀死敌对的单位来获得经验。敌对的单位包括敌对玩家和野生单位。

杀死单位时没有英雄在附近，那么你的英雄同样可以得到经验（这是为了鼓励使用空军来骚扰对手）；在团队游戏中，你的盟友同样可以得到经验。如果有一个英雄在附近，那么只有他能得到经验。

如果有2个英雄在附近，那么经验就平摊给2个英雄。

英雄可以通过摧毁攻击性建筑来获得经验，但不能从攻击性建筑杀死的单位那里获得经验。

你可以通过杀死掉线玩家的单位来获得经验。这在FFA（混战）游戏中十分有用。但是你不能通过摧毁他们的攻击性建筑来获得经验。如果你的英雄超过5级，就不能得到经验了，同野生单位一样。

如果玩家只有1个英雄（包括死掉的），那么在他的基地处于2、3级时可以得到经验的加成。2级时是15%，3级时是35%。这对敌对玩家和野生单位都有用。

英雄复活
尽管英雄有传奇色彩的身份，但是他们同样会在战场上死去。当英雄死掉时，不会留下尸体。当他的身体消失时，灵魂就回到了祭坛。每个种族都有祭坛来召唤和复活他们的英雄。复活英雄不需要木材，只要黄金。黄金的费用和英雄的等级有关：等级越高，花费越多。复活英雄同样需要时间。

  1. 鼠标停在需要复活的英雄的按钮上会显示名字，等级，英雄类型和费用。
  2. 祭坛复活的英雄拥有100%的生命值和100魔法值，而酒馆唤醒的英雄只有50%的生命值和0魔法值。

介绍
英雄与普通单位的另一个区别在于英雄有属性。这3个值会随着英雄的等级而变化。它们直接影响英雄的移动，攻击等能力。英雄升级时，属性会自动提升，使得英雄更加强大。英雄也可以通过物品来提升属性。

属性增加
当英雄升级时，属性点数会自动增加。

每种英雄类型的英雄在属性提升上是不同的，即使同类型的也是有差别的。

注意英雄的属性增加总点数，你会发现英雄一般都是6点，但也有不是的，火焰领主6.1、地精炼金术士6.3、巫妖6.4，知道我想说什么了吧！

主要属性 按照英雄的类型，每个英雄都有一个主属性。每提升英雄的一点主属性，都会增加他一点攻击。

力量
力量是物理能力的量度。

1点力量 = 25点生命
总生命值 = 100 + 25 * 力量 + 物品加成

1点力量 = 0.05生命恢复/秒
生命恢复速度 = 基础值 + 0.05 * 力量 + 物品加成 * 技能加成
基础值：
暗夜精灵族英雄：0.5（只在晚上）
不死族英雄：2.0（只在荒芜上）
其他英雄：0.25

智力
智力涉及到英雄控制魔法力的能力。

1点智力 = 15魔法
魔法值 = 15 * 智力 + 物品加成

1点智力 = 0.05魔法恢复/秒
魔法恢复速度 = 0.01 + 0.05 * 智力 + 技能加成 + 物品加成 * （0.01 + 0.05 * 智力）
每个英雄的数据表里的魔法恢复速度为基础魔法恢复速度，即为0.01

敏捷
敏捷决定了英雄护甲值和攻击速度。

1点敏捷 = 0.3点护甲
总护甲 = -2 + 0.3 * 敏捷 + 0.3 * 敏捷加成 + 物品加成 + 技能加成
注意：英雄护甲公式对于英雄的初始护甲，稍有偏差。

1点敏捷 = 减少 2% 攻击间隔
攻击间隔 = 基础攻击间隔 / （1 + 0.02 * 敏捷）+ 物品加成 + 技能加成
每个英雄的数据表里的攻击间隔为基础攻击间隔

技能加成：比如专注光环给予的护甲加成、辉煌光环/魔法泉对魔法恢复速度的加成、嗜血/耐久光环对攻击间隔的加成、邪恶光环/生命之泉对生命回复速度的加成。
